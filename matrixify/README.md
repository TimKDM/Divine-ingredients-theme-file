# Matrixify import files

Generated by mapping the client's "By Type" and "By Function" spreadsheets
against a fresh Matrixify product export from the live store.

## Files

| File | What it does |
|---|---|
| `products-update.csv` | Tag-only update for 36 existing products. `Command: MERGE` + `Tags Command: MERGE` so existing data is preserved. |
| `collections.csv` | 17 smart collections (5 By Function, 7 By Type, 5 category-level). Auto-populate from product tags. |
| `menus.csv` | Main menu structure: Shop by Function submenu, Shop by Type submenu, plus top-level Powders / Spices / Founders' Favourites. |
| `unmatched-products.csv` | 11 products listed in the spreadsheets that don't currently exist in the store (new products to add). |
| `products.csv` | Original full product file from the first import. Kept for reference. |
